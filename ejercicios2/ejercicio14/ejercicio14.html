Preguntas/Repuestas

¿Qué es una variable en programación?

Una variable es un espacio de almacenamiento en la memoria que tiene un nombre simbólico y se puede utilizar para almacenar datos.

¿Qué es un bucle?

Un bucle es una estructura de control que permite repetir un bloque de código varias veces.

¿Qué es un array?

Un array es una estructura de datos que permite almacenar múltiples valores en una sola variable, utilizando un índice para acceder a cada valor.
