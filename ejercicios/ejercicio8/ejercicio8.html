Palabras que se utilizan en el ámbito laboral

home office:
Trabajo desde casa, en lugar de ir a la oficina.
Base de datos:
Conjunto de datos organizados y almacenados de manera estructurada.
Big data:
Conjunto de datos tan grandes y complejos que requieren de herramientas especiales para su procesamiento.
Linux:
Sistema operativo de código abierto, utilizado principalmente en servidores y sistemas embebidos.
HTML:
Lenguaje de marcado utilizado para crear páginas web.
CSS:
Lenguaje de estilo utilizado para describir la presentación de un documento HTML.
JavaScript:
Lenguaje de programación utilizado para crear interactividad en páginas web.
